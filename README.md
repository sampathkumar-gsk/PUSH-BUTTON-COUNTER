# PUSH-BUTTON-COUNTER

## Project Description
This project uses Arduino with a Temperature Sensor.  
The temperature sensor reads temperature data and displays it on the Serial Monitor.

## Components Used
- Arduino UNO
- DHT22 Temperature Sensor

## Software / Platform Used
- WOKWI
- GitHub

## Working
- Temperature sensor reads temperature values
- Temperature data is displayed on Serial Monitor

## Output
- Temperature values are displayed successfully on the Serial Monitor.
